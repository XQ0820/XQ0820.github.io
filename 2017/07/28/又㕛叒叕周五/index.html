又㕛叒叕周五

  一进入周五，莫名其妙的兴奋，私心想着终于解放啦！周五晚上如果不加班一定是最开心的时候，没有之一。网上做调查一周7天中最喜欢的是哪天，必须周五，有工作带来的烦恼，同样也有对马上放假的期待，也是没出息，工作辣么久，追求的始终是有个开心的周末。

  给自己下班后的三小时安排了一项轻松地英语学习任务–看美剧，没法预测后面学习成果会怎么样，一想到能看美剧了，回家走路都带着风，恨不得哼着调调踩个风火轮。互联网行业大多数公司里面的每个人几乎都得背上多多少少的KPI指标，指标不同大体都是希望员工一心一意为公司奉献，研发自然不能例外，大家一股脑疯了一样连续加班好长一段时间。公司之前规定项目按时完成给每个人发数额不同的奖金，后来融了钱公司规模变大后，不知道哪位领导看不惯此项奖金改了规定，于是说好的奖金成了遥不可及的星星。本来也没怎么发过奖金自然不会被这不开心的规定影响心情，只是作为底层员工对领导层的决定经常摸不到头脑。

  周五工作不是特别多，中午吃过饭把关注的公众号未读文章翻了一遍，有个作者对于《2000万在假装生活》的评论特别好看：承认自己混的不好能怎样呢，你月薪一万感觉焦虑，是打开方式不对，换成2000块基本工资8000块房租看你还焦虑不焦虑。08年我还在读大学，作者说但凡早点来北京的青年才俊那几年前后手里有个几十万存款，买套北京房子一点都不难，即便如此还是有人选择将存款高息存入银行10年、20年，存款到期后，钱不值钱了，房价涨上天了。

  你问，现在买房来得及么？晚了!

  车撞墙知道拐了，房价上天想买了，鼻涕流到嘴里想要甩了，犯罪杀人蹲监狱知道悔改了，社会怎么会等你回过神来再给你一个机会？这段话简直说道心坎里。扪心自问，即使回到08年，有足够经济自主能力，自己可能也不会很乐意提前好几十年背上房贷，要知道工作后有的第一笔存款，自己首先做的是存进银行！定期！说到底是个眼光问题，08年淘宝开店正火时，有人觉得淘宝店已经过了气候，无论人力财力投入都比早期多很多，然后想一想便转身投身其他行业，偏偏一大批网红店正是从那时候开始，比如国民老公前女友雪梨。没有眼光和前瞻意识，即使黄金就在脚踩的这片土地下，你也不一定会捡得到。

  据说2000万那篇文章在阅读量超过2000万后被删除了，数字是否对得上无从得知，如果属实倒有一点讽刺意味。同篇文章不同人看有不同解释，自己读觉得原作有点帮老北京叫屈，而大多数人关注到的竟然是文章里面提到的5套房子，别人有一套房子，你嫉妒；而别人有5套，你只会觉得那生活太遥远，我们会嫉妒的都是自己跳跳脚能够到的高度，谁都不会随便定个成为王健林二代的小目标。听说万达文旅项目出手后，国民老公许多“老婆”纷纷倒戈，大家跟着形势走的速度还真是眼一睁一闭那么快。不能成为王健林第二，甚至也不能活成王可可，往常加班会说累成狗，其实狗没那么累，看看老公给他的COCO是什么待遇。

  除掉一如既往地对周五的爱，给这个周末加料的是到账的理财产品，收益不多只有几百块，钱不多可出去浪总有点底气，尝到了甜头想让甜更持久，国家新政要求理财产品仅面向风险评估合格的客户出售，各银行对未购买过理财产品的用户进行了不同提示。几乎每次出什么新规，便能多多少少看出来国家支柱产业效率有多低。互联网如此发达情况下，还有许多银行要求客户必须到网点进行评估，尽管城市银行在客户服务上比国有银行好很多，但仍然是低效率。某国有大型银行打电话问个问题，都很不耐烦的回复，吓得你以为自己问了什么不该问的东西，恨不得马上赔礼道歉耽误对方时间。五花八门的银行客户端，完全不考虑用户体验，找个什么东西要在界面上翻好几遍。。。个人最喜欢并且使用最久的招商银行也不那么友好，界面上好大一个提示需要到网点进行首次评估。

  考虑到银行可能是为了保护客户隐私，然而去到某银行网点时，前台小姐姐直接给了一支笔俩张纸让我站在那里做题，旁边来来回回不停地人啊。。。上面首先要填写的身份证和银行卡号一系列隐私信息，想着小姐姐一会应该是会把资料放在档案袋之类的地方吧？然而直到离开银行，小姐姐也只是把俩张单薄的问卷丢在桌子上。互联网时代虽谈不上个人隐私，但看着写了各种信息的单子孤孤单单躺在那里，好紧张。他们不会转手把风险问卷卖给第三方吧，第三方会不会搜到我的住址，会不会有人在某个月黑风高的晚上躲在小树丛里等着抢我分期还没结束的手机。
