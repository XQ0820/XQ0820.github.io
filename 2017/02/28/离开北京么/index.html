离开北京么

   软软昨晚做了个梦，梦里面和家里人去国外旅行，恰好住宿的酒店遇见了备考的大学同学，并且与他们住在一层，同学之间不亲密只是寒暄着打了下招呼各自回房。过了不久，怀里抱着一团白胖胖的的东西，一个人坐在房门旁边哭着希望这个不知道是什么的小生命坚强活下来，家里人和同学循着哭声赶过来见状很莫名其妙，软妈拍了拍她肩膀，好像安慰地说没关系之后留着她一个人在廊道里面伤心到不行，奇迹是眼泪似乎被怀里面这一坨东西全数吸收进去，不断蜕化竟变成了一直透明的蝴蝶。不能理解梦传递的是什么，想到破茧成蝶也是对自己的激励，一清早心情格外好。

   翻开新闻，除了被炒上天的房价头条还有天津大悦城不小心跌落的俩个小朋友，新闻图片上的儿童脸部被打了马赛克，但画面上清晰地看见跪在地上的男人和躺在小身体旁边的妈妈，虽从未为人母，见到这样的情景也特别感伤，即使一只猫猫狗狗养足了年月，送去别人家寄养几天都会想念；这位妈妈却痛失怀胎十月的爱子，报道说母亲怀抱俩孩子，一个宝宝不小心坠落，妈妈伸手去抓然后怀里的另一个宝宝跟着掉下去了。事发于商场四层饭店旁，坠落在负一层，可能这对夫妻本来是想带着一双子女享受一下工作之后难得的休闲，不想酿成了悲剧。评论区大多是感慨俩个孩子没有好好看过这个世界，还有一些说说风凉话，软软却很担心这位母亲的心理状态，她应该会背负着一辈子的内疚和怨恨吧，或许灵魂也跟着俩个亡故的小生命去了，即便身体允许，可以再次生育，这对夫妻会不会再也没勇气过幸福生活，希望她能尽快的走出悲伤，俩个小宝贝在另一个世界快乐成长。

   北京这种超级城市节奏太快，死了再多的人，在大多数眼里不过是多了几句感慨，谁都不会停下来；听闻昔日公司同事离婚了，一个素昧平生的女孩子朋友圈晒出了离婚证；真是七荤八素十载，一纸凉薄情不在。另外的同事之前一直很羡慕她每天都有老公接送，嫁的北京人，俩人从学生时代恋爱，走过了毕业分手季，走过了七年之痒相看俩烦，去年还一脸幸福的想当个准妈妈，今年就得挺着高烧，独自住院。女人之间，这种事不免成为八卦的好话题，大家你一嘴我一嘴，各抒己见，整体来说都是为这个女孩惋惜，将最好青春给了你，最后落得四海为家。晒出来的决心书上，提到了另一个女子，应当是两个人之间矛盾直接点，软软觉得她那封手写的信大概哭完了一年的泪水，所有人都为她打抱不平，还好姑娘看起来状态不错，比从前爱穿爱打扮；这种小三上位，逼走原配的桥段电视剧里面常常播放，真的在身边发生对于软软这些三观很正的良好女青年还是很难接受。

   从前老人讲，一个巴掌拍不响，相信姑娘一定是经过了很多夜晚不眠头疼欲裂的折磨和内心百转千回的挣扎才做出了最后的决定，信里面的负心汉也是绝对过错方。但如果是朋友，软软真想坐在她身边也请她想想这段婚姻里有没有哪些做的不好的地方，比如为什么结婚之后放弃护肤放弃时尚，一心一意扑在一个男人身上，硬生生将媳妇熬成婆，先生是惯犯成瘾还是偷腥，发现问题后，坐下来聊过么。一辈子真的太长了，软软始终不相信俩个小小的红本本能困住一颗骚动的心，男人也好女人也好谁都不可能一辈子忠贞一人，只是诱惑出现时看你是否欲望大过了责任。

   不幸的家庭各有各的不幸，听着别人的故事过自己的生活，想到和先生周末频繁升级的吵架，理由都是钱钱钱。软软开始认真思考，是不是到了该离开北京的时候了，想着也许永远买不起的房子，或者买了就要一辈子的房贷，还有遥遥无期的车牌，未来宝宝的升学，一桩桩一件件没有可以停止焦虑的。一间小房子装了多少年轻人的梦想，她还能记起来自己多想有个属于自己房子的心情，想要周游世界踏遍山川河流的愿望，这一切最终都得为了一幢小到装不下俩张床的屋子折腰。所以读书，拼命长知识最后都只是为了给这国家伟大的建筑事业添砖加瓦么，北京有它的魅力和接纳一切的宽容，却也有社会高低分层的残酷，每天坐三小时地铁，7小时觉，能算出来的时间几乎都给了办公室里一块小小的方格。

   北这座城市承载了太多悲欢离合，爱它也恨他，爱到想在这里生根发芽，恨到想一辈子都不愿意停下看看它；这里有其他城市不能提供的工作，有便捷的交通和不会被埋葬的能力；这里也有买不起的船票和上不了的车。是不是真的该考虑一下离开这里，哪怕只为了少一个理由吵架，少担心腰包几张可怜的人民币赶不上通货膨胀的变化。
