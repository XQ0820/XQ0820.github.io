大浪淘沙

   又要交房租了，辛苦赚几个钱一年到头留不下几个全给房东大人，每次交房租时，都会质疑留在北京的意义，不觉得自己是个完全的享乐主义者，确实会在每次发工资后留一小部分作为奖赏，但余下的所有部分通常在面对突如其来的疾病或者意外时显得微不足道。感慨生活不易时常会联想到母亲带自己的艰辛。

   亲戚们都以为小城镇出来在北京摸爬滚打不是人才就是精英反正有过人之处，听他们这样说会有点宽慰，只是心里会暗自嘲笑，不过比其他人多几分傻气。曾天真的想这么大的北京城怎么会没有自己的小小容身之所，可这个大城确实没有。每一个生活在城市的外地人就像海里的沙子，太细小太差都留不下，大浪淘的是沙最后却只留下金子。可能每个人都一样，不敢生病拼命赚钱成为留下的唯一动力，所有不太闪耀的愿望会被磨灭，会淹没在夜晚静谧灯光里。

   同事宝宝2岁半，到了上幼儿园年纪，他说在北京最多还能呆几年，这里没房没户口实在留不下。年纪每大一些身边离开的人便更多一些，甚至来不及一一道别。我们还扑腾在海水里，海水好咸，虽然同坐在一个办公室拥有同样蓝天，可有的人能放下手中工作随时起航，有的人战战兢兢生怕保不住饭碗；前者是有户口有房北京人后者是我这种用户画像。

   姐姐家小宝刚会爬，慢慢学习走路阶段，每次发来小宝视频，能反复看好几遍。小孩是上帝派来的天使，可爱调皮笑声如银铃。昨天先生做CT，一年内孩子事情不能考虑，凡此种种特别想逃避，能承受压力阀值不断变大，上一秒以为撑不过去，下一秒硬着头皮熬下去。大自然才是智者它巧夺天工的雕琢世界颜色，又惩罚每个不谙自然之法置身其中的子民，老话说，没有金刚钻别揽瓷器活，可生活在这个城市的人们似乎都在不断透支精神与健康，用痛苦乘以时间打败倔强的性格，非要拖到被逼得不行了才肯屈服。

   高中闹退学，家长们说多了一个大学证书能在初入社会时少碰棱角，至今非常感谢英明神武的老妈没放弃自己。可为什么读了书，生活在大城市，却还是这么多漂泊感越发想回家，绕一圈想做不读书也能做的事。为什么每个家庭都要拼命生小孩，都有皇位要继承？即使养儿防老在恶劣生存条件下能活到老那天么，手里攥着好像不少的钱用来准备老了以后去看病？

   跟先生说，在北京呆累了，想找地方安顿下来；先生每次都嗤之以鼻问：“你觉得回家能找到目前工作么？你能保持现在生活水平”，是啊，我能么，即使都能，哪里是家？讨厌他每次在我拿出一堆想法后，抛出一堆看起来没有任何破绽的问题，安顿下来至少有时间能吃早晚饭，况且现在哪里好，钱没多攒，衣服没多穿。处处无解，生活无望，只有仰起头的那片天才愿意接纳自己，死后不能长眠于地下，土地太贵，不如化一缕青烟飘在城市上空安静看苟且活着的另一些人。

   未来到底在哪里，活着要奔向何方。世界上没有一处可以安宁，这边想闭眼休息，那边讨论车房居住证暂住证。魔鬼经济学书上说每个偶然结果后面都有个必然原因，10年以后一定会有专家跳出来分析我们这代人堕落的原因，或者在他们有力指导和研究下青年变幸福的有效措施，原因有国家大力限购，鼓励小城镇发展，全民创业等；其实最根本原因是10年前房价太高年轻人生不起孩子，老龄现象严重加剧。

   You can be young only once, but you can always be immature. But for me, I can neither be young nor be immature.
