噩梦

   去年读英语以来，噩梦出现次数越来越少，年后公司换了个产品经理，每天都得东改西改几乎无一日安宁，那以后，噩梦又开始频频造访，百度给出了一系列可能原因，其实心里很清楚，是因为自己抗压能力需要提高，对每日层出不穷的问题心有余悸，心理压力在梦里变成了一个个可怕的小人，或者变成厉鬼或者拿着小刀；有的梦即使白天再复述一遍都觉得脊背发凉，好在梦里被刀割不会特别痛，不然要在梦里哭醒无数回。

   上上日梦见家对面住一位女鬼，有血有肉还对我笑，起初以为她是善意好心帮忙看门，后来房门关上，一扭头还能看见她坐在家里沙发旁边对着自己歪脑袋不停笑；于是在梦里也吓得半死的自己，穿着睡衣扑通一声跪在对面正在吃晚饭的邻居叔叔阿姨面前，哭着求她们说她们的女儿不是谁派来伤害自己的；先生对夜里常常说梦话的情况早见怪不怪，只要听见不停的又吼又叫会摇摇胳膊把我弄醒；为了不经常在梦里被各路妖魔折磨，俩人有个约定，只要发觉我正在做恶梦，无论何时一定要把人摇到头脑清醒。

   都说梦是现实生活的反射，有梦可做的夜晚，大多数是不愿意回想起来要么有张牙舞爪上下乱窜的厉鬼，要么是前有追兵后有悬崖的困境；这样看来，每日的生活是有多糟糕以至于小鬼们夜夜笙歌。噩梦做多了，顶多醒来时候擦擦汗，抓紧被子，先生不出差窝近他的臂弯；先生不在家，抓来猫猫放在头顶；麻木且淡然。如果有份适合普通人剖析当前心境的对照表，自己心态应该在最差一级，神经总想抻着的弹簧，稍不留神崩的四肢五脏皆疼。

   昨夜又梦，这次没遇到看不清脸头发老长老长，狂风骤雨瞪着铜铃大的眼睛的鬼怪，却遇到一位随身携带匕首的帅哥。梦发生在某个学校的操场上，里面每个人都满脸洋溢青春年少，自己一身白色运动衣绕着塑胶操场慢跑，后面一个男生气喘吁吁的也跟着跑，拐弯处几个箭步冲过来一把扑倒我，恶狠狠地问：“和你说的事考虑怎样了，到底要不要做他朋友？”？？？？梦里人设特别奇怪，我也不是啥世界富豪，做不做朋友既不能帮他赚钱也不能替他跑腿。还没等到回过神解答他问题，小帅哥从袖子里掏出一把褐色匕首，匕首刀尖特别短，尖尖三角型，三角俩侧都是利刃；他举起刀子毫不留情的在胳膊下面肚子旁边一一开了光，刚被割开皮肉痛感神经还没睡醒，只看见体内鲜血像汩汩小溪哗啦啦的从身体里奔涌着向前冲，我尖叫着喊周围人救救自己，那些本来都在看台上的同学们不知道几时起纷纷跑去离自己很远很远的另一侧，他们三五结伴，望去像家里面摆着的白色玩偶；应该是梦里喊得声音太大，吵到正在熟睡的先生，噩梦告一段落。

   大学住宿舍时，同寝舍友说我常在夜里喃喃有词的骂人，她们组团问我是否患有夜游症，还开玩笑说要把宿舍里面所有能作案的工具藏起来，毕竟夜游杀人不犯法，否则哪天稀里糊涂的被杀掉，留家里年迈父母无处喊冤。大学做了些什么梦早记不得了，可是梦多了会向串起来的长串电视剧，只不过剧中只有主角一人，其他常偷偷入镜头的总是些现实生活没见过的样貌。初次听说罗琳一梦写出一系列经典的《哈利波特》，特感慨啥时候自己能做罗琳第二，可惜第二没当成，七七八八都是荒诞诡异的噩梦。

   或许这些日子梦里那些人，是取材于马亲王的《文化不苦旅》，这本在读书籍已经接近尾声，书中写到诸葛丞相六出祁山最后一站五丈原，亲王游记精彩处是娓娓道来的一路行进像情结丰富的小说，读的人欲罢不能；“出师未捷身先死，长使英雄泪满襟”，上学时候学古文不很清楚每句含义，这次借着书中先生对丞相的描述，豁然开朗。丞相殁于此地，积劳成疾，后身尚有未竟的克复中原大业，亲王书中写到如果丞相当年能多关注自己的身体，或者放手一搏，中国的历史可能会被改写。性格决定命运，丞相一生谨慎，此结局也不出人意料，只遗憾千古一人，闭眼时都没能完成自己的愿望。丞相走前应该也日日噩梦吧，据说人在临终前多少有些感知，不知道他临行前一晚有没有睡一个好觉。
