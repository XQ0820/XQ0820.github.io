马桶沉思

   如果作息时间规律，软软到公司后的第一件事是处理“内部矛盾”，坐在电脑前面时间太长了，加上经常饮食不注意，肠胃落下很多小毛病，坐在马桶上不能痛快顺畅，公司的卫生间朝北，冬天没有暖气空调又不暖时，坐上去之前得有点心理准备。马桶这种坐着方便的东西流行了好多年了，设计者的初衷应当是为了减少蹲坑时候双腿的酸麻感，可是软软却不太喜欢这种设计，第一次用马桶时候本来翻江倒海的那些脏东西在大腿接触到冰凉的瓷砖时全无，而且总觉得使不上劲，不舒服。令人庆幸的是，无论达官贵人，土豪绅士坐在马桶上都一样不会有尊贵与位置的分别，谁都得先退掉屁股上面那层皮。听说日本的马桶带很多智能功能，特别受国人喜爱，许多人出门旅行都不远千里运回一个马桶盖，不知道那种先进的盖子放在她们这样终日没有阳光的卫生间里会不会温暖一些，有没有哪个高级公司不惜高价已经为员工提供了这样隐形的福利。

  国内屡屡爆出来的各类食品，化妆品，等等问题，让大家对国外的产品趋之若鹜，好像国外不会买到假东西一样。软软工作几年，手头稍微有点零钱总得流连于各类电商网站，从牙膏洗发水到服装包包；最近正好一款化妆品用完了，赶紧去某APP上下单，一来二去买了好多计划外的东西，账上的余额biu一下没了一半，看着短信通知账单肉疼的要死。她常常安慰自己，女人总得对自己爱惜一点，让胶原蛋白待在脸上时间长点，况且现在年纪不大正当花容月貌的好时候，先生又不反对，何不趁大好年华享受享受。20多一片的面膜不便宜但比起房价也确实不贵，有小伙伴为了孩子老公和房贷吃穿用度基本都要选性价比最高的，然后眼睁睁看着他们身材日益走形，也很不理解她们拿着高薪为什么把自己整成这个模样，但看着家家开着车买了房又很怀疑到底怎么做更对。那些成了家的女人婚后一心为家庭，似乎家里的每个人都值得照顾独独除了自己，孩子还小要吃好穿好，老人年纪大了要玩好用好，老公事业发展得买辆好车，自己都当了妈无所谓再折腾。为她们的牺牲精神点赞，可实在做不到像她们这样节俭，软软感觉现在生活中的很大乐趣全在于平日看上喜欢东西能买买买，要是把这部分全都砍了去，什么都得考虑性价比，生活实在无趣。她很清楚，不认同这些人也不代表自己完全对，有些东西还是该学习，像怎么合理有效的将为数不多的存款保值或者翻倍，怎么在生育前保养身体。

  随着国内人出国旅行人数增多，中国人的消费能力变成了许多国家旅游经济的重要支柱，韩国日本同属亚洲，假货少且方便成了许多人迈出国门的首选，软软也是忠实的韩货迷妹。可韩国首脑好死不死的搞出个什么萨德事件，侵犯祖国领土权益，令一些人坐不住了，我们国家我们怎么骂那也是亲妈带的亲娃，别人算个老几指指点点的，还敢骑到头上示威，老虎不发威当我们13亿人是病猫呢。政治事件不多做评论，国家层面的决策并非是软软这种平民之辈可以揣测，有些事读了再多书也不一定看得懂。可韩国人还要赚钱，在这片黄土地上有他们的生意，天朝民众看不懂国家动作，但可以去他们在这里开枝散叶的超市胡闹，于是苦了在韩资企业上班的国人。去卫生间前，同事找他讨论早晨的热门新闻，某区群众在乐天玛特超市前拉横幅抗议，实在可悲又可笑，大家团结在一起真的要抵制某一个国家的产品，对他们的销售额一定是个不小的冲击，可除了这些热心的大妈还有很多人就是认为国外的月亮更园，马桶盖更软，难到要绑住所有人的腿？

  不过以目前的思维和智商，除了绑腿也确实想不出好办法，不禁想起伟大作家鲁迅先生当年弃医从文，为的是从精神上唤醒国人，软软也可以试试这个办法。叫醒别人之前得先让自己清醒，她最近满腔热血的文章基本全是自嗨，没主题没中心，到了末尾会忘记开头写的什么，书也读了一些，可停在脑袋里不过几秒钟就变成了跳跃的催眠音符，看人家一气呵成的文章真是羡慕，那幽默的措辞天马行空的想法实在非一朝一夕能补，成为万古流芳的作家道路太远啦，得踩着前面无数无数人的头还得比后来居上的人勤奋且优秀。。。还是由着大妈们来吧，等她有洗脑的能力估计地球得由东向西走。权当自己日日哼哼唧唧的文字是个人的一种长线投资，别让时光白白流逝，老的时候有个可以翻出来的东西供消遣，可能哪天真被趣味相投的有钱人看中，从此一骑绝尘奔赴戎马倥偬的文字路。

  马桶上面想的东西不作数，凉了大腿还冷了心。
